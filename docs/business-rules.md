# Business Rules

## 活動限制

1. 訂單號碼不可重複使用
2. 即使活動結束也不可再次使用
3. 一位使用者一天只能有一筆訂單號碼進行中活動
4. 完成三個定點打卡才能兌換獎品進行核銷

---

## 活動時間

1. 活動開始時間為建立時間
2. 活動結束時間為次日 00:00

---

## 活動狀態

| 狀態     | 說明           |
| -------- | -------------- |
| 未完成   | 尚未完成景點   |
| 已逾期   | 超過活動期限   |
| 未領獎   | 已完成但未領獎 |
| 已領獎   | 已核銷         |
| 獎品待補 | 已完成但缺貨   |
